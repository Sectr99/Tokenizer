# Changelog

All notable changes to Token Forge will be documented here.

---

## [1.0.0] - 2025-06-14

### Added
- Initial release
- 9 built-in procedural token borders: Iron Ring, Gold Signet, Blood Pact, Arcane, Nature, Spectral, Infernal, Draconic, Ancient Stone
- Portrait upload via drag-and-drop or file browser (PNG, JPG, WEBP)
- Drag-to-reposition portrait on canvas
- Scroll-to-zoom on preview
- Independent X and Y scale sliders with linked/unlinked toggle
- Reset position & scale button
- Custom border upload (PNG, WEBP)
- 512×512 transparent PNG export
- Dungeon-themed animated background (stone wall, torch glow, embers, smoke, dripping water)
- Fully self-contained — single HTML file, no dependencies
